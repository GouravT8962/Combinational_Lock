This is the Readme file for Combinational Lock

It accepts 4 digit serial input. 
It has 3 buttons:
a) Reset button
b) Done button
c) Password change button

Please refer video before reading the steps to do various things

Steps to reset password:
- press button 7 , password is reset

Steps to open the lock:
- Serially press the correct lock combination and press button 5, lock will open otherwise it will be in lock state only

steps to change passsword
- press toogle switch 6
- then type current password, if password is correct, then it will continue otherwise goes to lock state
- if password was correct, then type new password
- toggle switch 6 and password is changed
